What is an MLB game simulator?

An MLB game simulator plays out a single matchup using team and pitcher data and returns a projected winner, a score range, a win probability, and a box score. The TrustMyRecord game simulator runs each game one plate appearance at a time so the box score emerges from real simulated events.

How do I simulate an MLB game?

Open the MLB Simulator, pick the away and home team, choose each starting pitcher, then run it. You can run one game for a quick read or run up to 100 simulations for an aggregate picture of how the matchup tends to play out.

Does the starting pitcher change the simulation?

Yes, a lot. Starting pitching is one of the biggest drivers of a game projection, so swapping a starter in the dropdown changes both the expected runs and the win probability. Comparing starters is one of the most useful things the game simulator does.

Can I bet based on a game simulation?

A simulation is research, not a guaranteed pick. Use the projected winner and win probability as one input, compare it to the real market price, factor in lineups and weather, then decide. When you commit, lock it as a verified pick so the result counts.
