How Arena rankings are built

Every ranking on this board comes from confirmed head-to-head console matches played through TrustMyRecord Arena. When two players finish a challenge or league game in MLB The Show, NBA 2K, Madden, EA FC, or NHL, the reported final score only counts after the opponent confirms it or an admin verifies it. Nobody can type in a fake record: wins, losses, win percentage, and streaks are computed from that confirmed match history alone.

Use the filters to rank players by total wins, win percentage, or current hot streak, and to narrow the board to a single game title. Rankings update automatically as new results are confirmed, so the board always reflects the current state of competition.

Want on the board? Accept an open challenge, join a league, or watch matches live on the Arena streaming page. New to Arena? Start with the Arena hub or the MLB The Show hub, and see season-long stat competition in the MLB The Show Stat League.
