# Find Handicappers Table Standard

The public `/handicappers/` leaderboard table must stay focused on the core stat columns:

- Username
- Record
- Units
- ROI
- Win %
- Picks
- Streak
- Active

Do not reintroduce a table-level Sports column or repeated sport badges. The sport selector belongs in the filter controls only.

The desktop stat columns must be spaced wide enough that records such as `58-54-3`, units such as `+4.90`, ROI, win percentage, pick counts, and streaks do not visually run together. Mobile may stack rows into labeled blocks as long as the same core stat set remains readable.

Leaderboard pagination must apply after search, sport filtering, and sorting. The default view is Top 25, with page-size choices for Top 25, Top 50, Top 100, and All, plus Showing X-Y of Z qualified members, Previous/Next buttons, and a Page N of M indicator. Search, sport filter, sort, and page-size changes must reset to page 1.
