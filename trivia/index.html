How Trivia Scoring Works

Points = base points x speed multiplier. Base is set by difficulty (easy 100, medium 200, hard 300). The speed multiplier runs from 2.00x for an instant answer down to 1.00x at the buzzer. Wrong answers, expired questions and forfeited questions all score 0. The clock and the scoring run on the server.

Stay on the page. Switching tabs, minimising the browser, moving to another window or navigating away while a question is live forfeits that question for 0 points.
